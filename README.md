# LUXXPOOL Emulation V3

Next.js recreation of the LUXXPOOL v0.8.0 mining pipeline emulation dashboard for Vercel deployment.

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub and import the repo into Vercel. Framework preset: Next.js.

## Stack

- Next.js 14 App Router
- React 18
- TypeScript
- Inline styles (no Tailwind needed)
